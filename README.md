This repository contains a simple Spring Boot application developed. The project demonstrates core backend concepts, cloud deployment on Google Cloud Platform (GCP), and containerization using Docker.

## 🚀 Features
* **Spring Boot Backend:** A lightweight REST API built with Java 17.
* **GCP Cloud Deployment:** Hosted on a Google Cloud Compute Engine (Ubuntu VM).
* **Dockerization:** Fully containerized application for environment-independent deployment.
* **Monitoring:** Resource usage tracked natively via Docker stats.

---

## 🛠️ Tech Stack
* **Language:** Java 17
* **Framework:** Spring Boot 3.3.4
* **Build Tool:** Maven
* **Infrastructure:** Google Cloud Platform (GCP)
* **Containerization:** Docker (Base Image: `eclipse-temurin:17-jdk-jammy`)
